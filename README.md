# Perched

Discover and share the best spots around campus. Coffee shops, libraries, coworking spaces and more.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

From the output, you can open the app in a development build, Android emulator, iOS simulator, or Expo Go.

Edit files inside the **app** directory. This project uses file based routing.

## Reset project

```bash
npm run reset-project
```

Moves starter code to **app-example** and creates a blank **app** directory.
